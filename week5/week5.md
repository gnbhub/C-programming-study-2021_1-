# 제어문 실습_0512

### ✅ 최대공약수 계산하기
유클리드 알고리즘 사용하여 최대 공약수 계산하기 <br>
**유클리드 알고리즘이란?)** <br>
```
㉠ 두 수 가운데 큰 수를 x, 작은 수를 y라 한다.
㉡ y가 0이면 공약수는 x와 같다.
㉢ r <- x % y
㉣ x <- y
㉤ y <- r
㉥ 단계 ㉡으로 되돌아간다.
```
<br>
- 입력 받은 두 수 중<br>
  · 큰 수 : a<br>
  · 작은 수 : b<br>
<br>

실행예시>>>
```C
정수 1 : 15
정수 2 : 12
  
최대공약수는 3입니다.
```
